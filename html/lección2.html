Las etiquetas del encabezado se utiliza para dividir el documento en secciones y subsecciones. El primer encabezado h1 es generalmente utilizado una vez, ya que este contendra el titulo del documento. Este a menudo contendra el mismo contenido como el <title></title> elemento. De forma predeterminada, cada elemento del encabezado es para indicar el estilo que va disminuyendo entre los diferentes encabezados. El h2 es más a menudo utilizado como título del contenido, por lo que es el subtítulo y dándole menos importancia que h1. El h3 recibe mucho menos, y así sucesivamente. La cadena decrese hasta llegar al h6. Dado que la información no es generalmente dividido en sub-sub-secciones y más allá, h4 y por debajo de las etiquetas son menos utilizadas que las de líneas de mayor encabezado.

Validación

El World Wide Web Consortium (W3C) proporciona un sitio web para comprobar validación para su página web. El URL por el validador es http://validator.w3.org . Usted puede comprobar sus páginas para ver si estan correctas como copiar y pegar, subir, o entrando una dirección web para ellos (si están en un sitio web público).

La barra verde significa que la página ha pasado la validación, y la barra roja significa que contiene errores. Usted debe revisar todas las páginas a medida que avanza a través de este tutorial y asegurarse de que reciba una barra verde.

  1. Copy pagina_basica.html a gdw.html .
  2. Ponga Manos a la obra HTML en el título y un nivel superior del título (h1) en el cuerpo.

  5. Utilice el validador del W3C para validar las páginas web que ha hecho en esta lección y en la lección 1. Asegúrese de obtener una barra verde en ambas páginas.
