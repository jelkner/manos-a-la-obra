Lección 6: Enlaces

Formato

Los enlaces son una parte esencial de un sitio web, ya que con un simple clic del ratón le permiten transferirse a cualquier otra página en la web.

El formato de un vínculo parecido a éste:

 <a
href="index.xhtml">Volver al inicio</a>

Veamos cada una de las partes de esta declaración:

<a ...>
es la etiqueta de apertura de anclaje.
href="index.html"
define un valor par atributo. El atributo en este caso es href, que se sitúa como referencia de hipertexto. index.html es el valor del atributo href, que se asigna, o destino, con el atributo por el operador d e asignación (=).
Volver al inicio
es una etiqueta en la que el usuario haga clic para seguir enlasado.
</a>
es el ancla de la etiqueta de cierre.

Attributes

Información adicional se puede dar a un elemento HTML usando atributos. Los atributos se asignan (o unidos a) los valores el operador de asignación (=). El valor está entre comillas(""). La sintaxis general es la siguiente:

atributo="value"

Hipertexto

El atributo en la etiqueta de anclaje, href, es sinónimo de hipertexto de referencia. Es el href atributo que convierte el elemento de anclaje en un hiperenlance (o como más comúnmente se llama, enlance). Los enlances son lo que impulsa World Wide Web y la hace tan popular.

Ubicación

El valor de un atributo href es por lo general una dirección web, que describe la localización de otra página web. Las direcciones Web también se conocen como URLs. Hay dos maneras de especificar la ubicación de un enlace de hipertexto:

Ubicación Absoluta

Una ubicación absoluta usa por completo el URL, ejemplo: http://openbookproject.net/tutorials/manos/xhtml/index.xhtml. Esto no solamente tedioso para escribir, pero con Ubicación Absoluta hace difícil de mover su sitio web, que nos lleva a ...

Ubicación Relativa

Digamos, que quieres ir de

http://openbookproject.net/tutorials/manos/xhtml/lección6.xhtml.

a

http://openbookproject.net/tutorials/manos/xhtml/lección7.xhtml.

Desde el interior de la lección6.xhtml archivo, el enlace sólo se debe escribir asi:

 <a
href="lección7.xhtml">Lección 7</a>
