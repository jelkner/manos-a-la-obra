Lección 9: Próximos Pasos

Hay una serie de lugares en la web para ir a obtener más información sobre XHTML y CSS. La siguiente lista de anotacion contiene algunos de los mejores:

http:tutoriales xhtml y css
Este sitio cuenta con excelentes tutoriales sobre una amplia gama de dominio de web tema. La XHTML Tutorial y la CSS Tutorial Tutorial son los mejores lugares para comenzar futuros estudios hacerca de estos temas.
http://www.blooberry.com
Blooberry esta bellamente organizado para ambos html y css. Es una referencia, no un tutorial, y el sitio está empezando a ponerse un poco viejo, pero sigue siendo un buen lugar para ir si usted sabe lo que esta tratando de hacer y sólo necesita algo de ayuda con la syntax para hacerlo.
http://www.webheadstart.org
WebHeadStart.org tiene tutoriales para HTML, XHTML, y XML. El sitio se encuentra actualmente en beta, pero la información que aparece esta bien presentada.
http://websitetips.com
Según su sitio web, WebsiteTips.com es un lugar cuidadosamente seleccionado, con una extensa lista de más de 2.400 recursos anotados. Los datos en ambos XHTML y CSS son realmente impresionantes.
http://www.csszengarden.com
CSS Zen Garden es el lugar para visitar si quieres ver lo que los maestros y artistas web puedes hacer con CSS para hacer un sitio web hermoso.


Ejercicios:

  1. Visite algunas de las paginas anteriores. Haga una pagina web en un archivo nombrado onmyown.html y haga una lista ordenada de 5 cosas específicas que usted encuentre en su búsqueda que no se a tratado en este tutorial. Localize la fuente de cada búsqueda e incluya un enlace a ella. Como siempre, asegúrese de que su página sea valida tanto para XHTML 1.1 como para CSS 2.1.

  2. Crear una página web en un archivo llamado miraquepuedohacer.html. Use todo lo que a aprendido hasta ahora, diseño de una página que le permite puntualizar sus cosas. Puedes pensar en esta página como la que se desea mostrar a un empresario si se esta aplicando para un trabajo como webmaster.
